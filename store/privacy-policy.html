Privacy Policy

Claude Usage Tracker — Chrome Extension

Data Collection

This extension does not collect, transmit, or store any user data on external servers. There are no analytics, no tracking, and no telemetry of any kind.

How It Works

The extension reads cookies from the claude.ai domain using the Chrome cookies API. These cookies are used solely to authenticate requests to the Claude.ai usage API on your behalf. The extension calls:

https://claude.ai/api/organizations/{orgId}/usage

This is the same endpoint your browser uses when you visit the Claude.ai usage page. The extension simply automates this request.

Data Storage

All data is stored locally on your device using chrome.storage.local. This includes:

  • Usage data returned by the Claude.ai API
  • Your card display order preference
  • Notification threshold flags

No data is ever sent to any third-party server, analytics service, or external endpoint.

Cookie Usage

The extension reads your claude.ai session cookies to authenticate API requests. Cookies are:

  • Read from the claude.ai domain only
  • Sent only to claude.ai API endpoints
  • Never stored, logged, or transmitted anywhere else

Permissions

  • cookies — Read claude.ai session cookies to authenticate usage API requests
  • alarms — Poll the usage endpoint every 5 minutes
  • storage — Persist usage data, card order, and notification state locally
  • notifications — Display desktop alerts when usage crosses thresholds
  • host_permissions (claude.ai) — Required for cookie access and API fetch
